Search NEMBASE4 for a specific sequence identifier

You can retrieve data from within NEMBASE4 using identifiers associated with the sequences.

In the search box below, enter either
• a cluster identifier (e.g. NBC00001 ),
• a protein translation identifier (e.g. NBP00001_1 )
• or the accession number of a nematode EST sequence (e.g. BM278826 ).

Then hit the 'Go' button to retrieve the details for that sequence.
